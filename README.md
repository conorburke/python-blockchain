
## Activate Environment 
```
conda activate bc
```

## Install Packages
```
pip install -r requirements.txt
```

## Run a Module
```
python -m path.to.module
```

## Run the Tests

```
python -m pytest backend/tests
```

## Start the API
```
python -m backend.app
```

## Run a Peer Instance
```
export PEER=True && python -m backend.app
```

## Run the Frontend
```
npm run start
```

## Seed the Backend
```
export PEER=TRUE && python -m backend.app
```
